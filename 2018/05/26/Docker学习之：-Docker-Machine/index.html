Docker学习之： Docker Machine

2018-05-26 07:13:49 | Docker

首先什么是Docker Machine呢？它是一个可以在虚拟机上安装Docker Engine 的工具，如果你是Mac,我们在第一节的时候就说过，我们在用安装包，安装完成以后Docker Machine就已经在我们的系统里面了，直接在命令行输入docker-machine version
如果你是win10的系统，Docker Machine也是默认安装好的，如果你不是win10也没有关系，你可以单独安装它，但是我们这里就不单独说明了。
如果你像我一样出现了下面的错误，多半是因为你没有安装virtualbox

我们在stackoverflow上面搜索这个错误，得到了一个回答, 请时刻记住，你不是孤独的，如果遇到问题，可以先在网上搜一搜。

当然你也可以下载安装包进行安装，如果你是从第一节看过来了，相信你的电脑上已经安装了virtualbox，那么就不会出现这个错误了。

  • 首先执行docker-machine create demo
  • docker-machine ssh demo
    #####如果你出现了下面的提示，就说明就安装成功了

    那么Docker Machine帮我们做了什么呢？它是帮我们在virtualbox上面安装了一个非常小的虚拟机并且已经为我们在这个虚拟机上安装了Docker，怎么样是不是还蛮叼的，你安装成功了嘛，开始我们的Docker进阶之旅吧！！！！！
